# تحديث تسليم 2026-06-10 - توافق صفحات التعلم العامة مع مشروع React القديم

## ما تم

تم تنفيذ دفعة توافق بين مسارات التعلم العامة في مشروع React القديم ومشروع Laravel:

- دعم رابط React القديم:
  - `/category/{path}?subject={subject}&tab=skills`
  - يتم تحويله الآن إلى صفحة المادة في Laravel مع الحفاظ على قيمة `tab`.
- صفحة المادة العامة أصبحت تقرأ التبويب المطلوب من الرابط:
  - `skills`
  - `courses`
  - `training`
  - `tests`
  - `library`
- دعم أسماء تبويبات React القديمة:
  - `foundation` يتحول إلى `skills`.
  - `questions` و`banks` يتحولان إلى `training`.
  - `exams` يتحول إلى `tests`.
- إضافة بطاقة "الخطوة التالية" في صفحة المادة حسب التبويب الحالي وأول محتوى متاح:
  - مهارة تأسيس.
  - تدريب.
  - اختبار.
  - دورة.
  - ملف مكتبة.

## الفحص

- `php -l routes/web.php` نجح.
- `php -l resources/views/public/subject-learning.blade.php` نجح.
- اختبار `PublicLearningCompatibilityTest` نجح:
  - تأكيد تحويل رابط `?subject=&tab=skills`.
  - تأكيد وجود منطق التبويب القديم وبطاقة الخطوة التالية في صفحة المادة.
- `npm run build` نجح بعد تحديث الواجهة.

## ملاحظات

- هذه الدفعة تقرب صفحات التعلم العامة من تجربة React القديمة، خصوصًا روابط `GenericPathPage` و`SubjectLearningPage`.
- ما زال مطلوبًا لاحقًا نقل تفاصيل أعمق من React مثل:
  - إرشاد الوصول حسب الباقة.
  - عرض موضوعات المادة بتقسيمات فرعية أوسع.
  - ربط ملفات المكتبة القديمة كنموذج مستقل بدل الاعتماد على `Lesson.content_url` فقط.
